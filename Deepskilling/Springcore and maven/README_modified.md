# Spring Core & Maven Hands-on

**Author:** Nikhil Singh Rathore

**Application Name:** `LibraryManagement`

**Package Name:** `com.librarymanagement.app`

This repository contains hands-on implementations covering the core concepts of the Spring Framework. It includes examples of Spring Core, Dependency Injection (IoC), Spring AOP, XML and annotation-based configuration, constructor/setter injection, Maven setup, and a simple Spring Boot application developed for practice.
